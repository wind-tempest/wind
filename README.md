Wind/Tempest is a barebones POSIX-like OS. Built from scratch, for amd64.

## Core Features

- **Boot:** Limine + Multiboot2
- **Display:** Framebuffer text & test patterns
- **Input:** Keyboard (US only for now)
- **Shell:** `inish` (basic commands)
- **FS:** Ext2
